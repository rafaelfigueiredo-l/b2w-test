### Getting Started
_______________________

```

$ Git --version
>= v2.25

$ node --version
>= v12.14.1

```

### How to run

```
$ git clone https://github.com/rafaelfigueiredo-l/bw2-test
$ cd bw2-test
$ npm install
$ npm start
```

###  How to build
```
$ npm run build
```

### How to test
```
$npm test
```
